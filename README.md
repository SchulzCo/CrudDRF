# CrudDRF

CrudDRF is a project developed with **Django** and **Django REST Framework (DRF)** that implements CRUD operations (Create, Read, Update, Delete) on a basic model.  
Its purpose is to serve as a template and practical example for building scalable and secure REST APIs with Django.

---

## Description

This repository demonstrates how to create a RESTful API using Django and DRF, with endpoints that allow easy data management.  
The project is designed as a learning base and also as a starting point for more complex applications that require authentication, permissions, and integration with external databases.

---

## Key Features
- Backend built with **Django 5.x**  
- API powered by **Django REST Framework**  
- Ready-to-use CRUD endpoints  
- **SQLite** database by default  
- Modular and scalable structure
